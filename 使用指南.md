# 🤖 悦海扬帆 AI 助手 使用指南

> **适用角色**: 跨境金融业务经理
> **运行方式**: Telegram Bot + GitHub Actions
> **核心价值**: 提升跨境业务处理效率 3-5 倍

---

## 📱 快速开始

### 第一步：在 Telegram 中找到你的 Bot

搜索 `@workbot`（或你的 Bot 用户名），点击 **Start**

### 第二步：使用命令

在聊天框输入 `/` 即可看到所有可用命令：

| 命令 | 功能 | 适用场景 |
|------|------|----------|
| `/start` | 启动并显示帮助 | 首次使用 |
| `/triage` | 业务分诊卡 | 收到客户咨询时 |
| `/policy` | 今日政策简报 | 每日晨会前查看 |
| `/checklist` | 材料清单+风险检查表 | 准备材料时 |
| `/meeting` | 会议速读卡 | 开会前准备 |
| `/draft` | 商务文档生成 | 写邮件/方案时 |
| `/log` | 项目进度记录 | 更新项目状态 |
| `/ask` | 业务问答（带风险审查） | 随时咨询业务问题 |
| `/status` | 系统状态检查 | 排查问题时 |

## 🎯 核心使用场景

### 场景1：收到客户咨询 → 用 `/triage` 分诊

**什么时候用？**
- 客户发来一个模糊的问题
- 你需要快速判断这个业务的类型、风险、优先级

**怎么用？**
```
/triage 客户想在哈萨克斯坦做ODI投资，制造业，预算500万美元
```

**你会得到什么？**
- 📂 业务分类：ODI
- 🌍 涉及辖区：哈萨克斯坦
- 🔴 风险等级：高/中/低
- ⚡ 紧急程度：紧急/关注/跟进/参考
- 📖 需查阅的政策清单
- ❓ 还缺少哪些关键信息
- ➡️ 建议下一步动作
- 👤 需要找谁确认

---

### 场景2：每日晨会用 `/policy`

**什么时候用？**
- 每天早上 8:30 自动推送
- 或者手动输入 `/policy` 查看

**你会得到什么？**
- 📰 今日最新跨境金融政策摘要
- 🔄 对不同业务路线的影响分析
- ✅ 今日建议跟进事项

**示例输出：**
```
📊 政策简报 (2026年5月10日)

🔥 重点政策更新：
• ODI备案新规：简化审批流程...
• 跨境结算：新增便利化措施...

📈 对业务的影响：
• 制造业出海 → 正面影响
• 资金出境 → 需注意新限制...

✅ 今日建议跟进事项：
1. 深入研究2026年境外投资ODI备案最新操作指南
2. 针对计划在哈国投资的客户，提供ODI备案具体要求
```

---

### 场景3：准备会议 → 用 `/meeting`

**什么时候用？**
- 客户要开会讨论某个项目
- 你需要在 30 秒内准备好会议要点

**怎么用？**
```
/meeting 香港公司注册咨询，客户是贸易型企业
```

**你会得到什么？**
- 📋 会议主题和背景
- 💼 客户画像（行业、规模、需求）
- 📌 讨论要点清单（按优先级排列）
- ⚠️ 风险提示
- 🎯 会议目标和建议结论
- 📝 会后行动项

---

### 场景4：写邮件/方案 → 用 `/draft`

**什么时候用？**
- 需要给客户发正式邮件
- 需要撰写商务方案
- 需要快速生成专业文档

**怎么用？**
```
/draft 给客户发邮件说明ODI备案流程，语气专业简洁
```

**你会得到什么？**
- 一封可以直接发送的正式邮件
- 包含专业的金融商务用语
- 结构清晰，重点突出
- 有明确的行动号召（Call to Action）

---

### 场景5：随时咨询 → 用 `/ask`

**什么时候用？**
- 遇到不确定的业务问题
- 需要快速获取专业意见
- 想了解某个法规的具体要求

**怎么用？**
```
/ask 哈萨克斯坦ODI备案需要多长时间？
```

**你会得到什么？**
- 专业、准确的回答
- 附带风险提示
- 相关政策参考
- 如果问题复杂，会标注"需人工审核"

---

### 场景6：记录项目进度 → 用 `/log`

**什么时候用？**
- 完成一个项目节点
- 需要更新项目状态
- 团队同步项目进展

**怎么用？**
```
/log 客户A的香港公司注册已完成核名，下周提交注册文件
```

**你会得到什么？**
- 结构化的项目日志
- 自动提取：客户、项目、阶段、里程碑、阻碍
- 同步保存到 Notion 知识库

---

### 场景7：准备材料 → 用 `/checklist`

**什么时候用？**
- 客户需要提交某项申请
- 你需要给客户列材料清单
- 需要做风险预检

**怎么用？**
```
/checklist 哈萨克斯坦ODI备案所需材料
```

**你会得到什么？**
- 📋 完整的材料清单（分类列出）
- ⚠️ 每项材料的风险提示
- 🕒 预计办理时间
- 💡 常见问题和解决方案

## 🔄 典型工作流

### 工作流1：处理新客户咨询

```
客户发消息
    ↓
/triage 快速分诊（30秒）
    ↓
判断风险等级和紧急程度
    ↓
高风险 → /ask 深入查询 + /checklist 准备材料
低风险 → 直接回复 + /draft 生成邮件
    ↓
结果自动归档到 Notion
```

### 工作流2：每日工作启动

```
早上到公司
    ↓
打开 Telegram
    ↓
查看 /policy 自动推送的政策简报（8:30已推送）
    ↓
标记今日需跟进的事项
    ↓
开始一天的工作
```

### 工作流3：准备客户会议

```
收到会议通知
    ↓
/meeting 输入会议主题（10秒）
    ↓
获得完整的会议速读卡
    ↓
打印或截图带着去开会
    ↓
会后 /log 记录会议结论
```

## 🔐 安全特性

- **数据脱敏**：所有敏感信息（金额、身份证号等）自动脱敏处理
- **风险审查**：每个回答都附带风险评估，高风险内容标注"需人工审核"
- **知识库归档**：所有交互记录自动保存到 Notion，方便追溯
- **权限控制**：只有你能使用，其他人无法访问

## 🆘 常见问题

**Q: 为什么有时候回复很慢？**
A: 系统每 15 分钟检查一次消息，最长等待时间约 15 分钟。

**Q: 回复的内容准确吗？**
A: 基于 GPT-4o Mini 和 Notion 知识库生成，但复杂问题会标注"需人工审核"，请务必二次确认。

**Q: 如何查看历史记录？**
A: 所有对话自动保存在 Notion 数据库中。

**Q: 遇到错误怎么办？**
A: 发送 `/status` 查看系统状态，或联系管理员。
